# АСБ РУМ ПРО

Строительная компания «АСБ РУМ ПРО» — строительство и ремонт от профессионалов.

## Структура проекта

```
├── admin/           # Панель управления Decap CMS
│   ├── index.html
│   └── config.yml
├── content/         # Контент сайта
│   ├── services/    # Услуги
│   ├── portfolio/   # Портфолио
│   ├── about/       # О компании
│   ├── contacts/    # Контакты
│   └── settings/    # Настройки
├── assets/
│   └── images/      # Изображения
├── src/             # Стили и скрипты
└── index.html       # Главная страница
```

## Управление контентом

Для редактирования контента перейдите по адресу `/admin/`

## Разработка

Сайт статический, для локальной разработки можно использовать любой HTTP-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```
